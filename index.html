使用言語・ツール

CSS
経験はHTMLと同じ程度、イラレやフォトショを使えないので出来る限りCSSで書こうとハック的な事をやるのが好き。左側のアイコンの下の星メーターもCSSとweb fontで出来てたりする。
HTML
大学に入って本格的に触り始め、とりあえず書ける。HTML5の機能はあまり使う機会がなかったので、まだ未熟。
AngularJS
Jqueryから乗り換え、バイト先のメインサービスの開発をやりつつ勉強した。一応動くものは作れるが、ディレクティブの設計などは甘め。
Jquery
HP制作を始めた際に独学で勉強し、バイト先のツールのV1で使用。最近はプラグインを利用する程度でコードは書いていない。
WordPress
フリーでwebサイトを構築した際に何度かお世話になった。カスタマイズ的なことは結構調べて実装してきたが、プラグインやSEO対策などは実践不足。
Rails
研究室内ツールなどを開発する際に使用。バックエンドの高速化やSQLに関して知識が不足しているので今は動くものが作れる程度。主にプロトタイピング用のJSONAPIを作るのに使用している。
